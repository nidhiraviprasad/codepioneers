Program Descriptions & Registration

Intro to Python

This is the perfect class for any student who wants to learn coding! Students will learn the fundamental building blocks of programming in the most popular language, Python. No prior coding experience required! Course material includes:

✓ Data Types

✓ Arithmetic Operators

✓ Variables

✓ Conditional Expressions

✓ Iteration

✓ Graphics with the Turtle Library

Register

Object Oriented Programming with Python

This course covers the fundamentals of object oriented programming! Students will write advanced programs using Python. Knowledge of the material covered in the Intro to Python and Intermediate Python courses is recommended. Course material includes:

✓ Objects

✓ Classes

✓ Inheritance

✓ Constructors

✓ Magic Methods

Register

Intro to Web Development

In this course, students will be introduced to web development and will learn to create websites using HTML and CSS. By the end of this course, students will be able to create beautiful and functional webpages. No prior programming experience required! Course material includes:

✓ Body tags

✓ Links and images

✓ Stylesheets

✓ Color, font, and other css properties

✓ Positioning properties

Register

Intro to Java

This course is the perfect introduction to the popular programming language, Java. Due to the complex nature of the language, we recommend that students taking this course have prior programming experience. Course material includes:

✓ Data Types

✓ Conditional Statements

✓ Iteration

✓ Variables

✓ Arithmetic Operators

✓ Arrays

What do I need to attend classes?

You will need a PC/Laptop with Zoom installed, as well as a stable internet connection. We will be using Zoom for virtual lessons, so a web cam is recommended. You will also need to read and agree to the terms of the Code of Conduct, Attendance Policy, and Media Release forms.

Summer Workshop: Intro to Python #1

This is the perfect workshop for any student who wants to learn coding! Students will learn the fundamental building blocks of programming in the most popular language, Python. Course material includes:

✓ Primitive Types

✓ Arithmetic Operators

✓ Variables

✓ Conditional Expressions

✓ Iteration

✓ Handling user data

✓ Graphics with the Turtle Library

No prior experience is required, as this is a beginner-friendly course!

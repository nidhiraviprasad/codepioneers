Program Descriptions

Summer Workshop: Intro to Python #1

This is the perfect workshop for any student who wants to learn coding! Students will learn the fundamental building blocks of programming in the most popular language, Python. No prior coding experience required! Course material includes:

✓ Primitive Types

✓ Arithmetic Operators

✓ Variables

✓ Conditional Expressions

✓ Iteration

✓ Handling user data

✓ Graphics with the Turtle Library

Offered twice a week for three weeks, and each class is 90 minutes long.

What do I need to attend classes?

You will need a PC/Laptop with Python and Zoom installed, as well as a stable internet connection. Python installation instructions can be found here. We will be using Zoom for virtual lessons, so a web cam is recommended. You will also need to read and agree to the terms of the Code of Conduct, Attendance Policy, and Media Release forms.

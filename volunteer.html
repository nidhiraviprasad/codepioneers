What will I need to volunteer?

You will need a PC/Laptop with Python and Zoom installed, as well as a stable internet connection. We will be using Zoom for virtual lessons and volunteer meetings. A web cam is required as well. You will also need to read and agree to the terms of the Code of Conduct, Attendance Policy, and Media Release forms.
